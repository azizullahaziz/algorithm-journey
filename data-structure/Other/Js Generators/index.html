JavaScript Generators Visualization

Understand how JavaScript generators work through an interactive visualization of the Fibonacci sequence generator.

What are JavaScript Generators?

Generators are special functions in JavaScript that can be paused and resumed, allowing you to produce a sequence of values over time.

// Generator function definition
function* fibonacciGenerator() {
    let a = 0, b = 1;
    
    while (true) {
        yield a;
        [a, b] = [b, a + b];
    }
}

// Using the generator
const fibGen = fibonacciGenerator();

console.log(fibGen.next().value); // 0
console.log(fibGen.next().value); // 1
console.log(fibGen.next().value); // 1
console.log(fibGen.next().value); // 2
console.log(fibGen.next().value); // 3
// ... and so on

Key Concepts

  • function* - The syntax to define a generator function
  • yield - Pauses the generator and returns a value
  • .next() - Resumes the generator execution
  • Generators maintain their state between pauses
  • Perfect for infinite sequences like Fibonacci
